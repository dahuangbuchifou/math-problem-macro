# 🧮 幼升小数学题生成器

> 专为幼升小儿童设计的 Excel VBA 数学题生成工具

一键生成加减法练习题，支持 100以内、两位数、三位数，具备难度分级、专项练习、A4 排版、多页生成、答案隐藏等专业功能。

---

## ✨ 功能特性

### 📊 难度分级（5 级）
| 难度 | 范围 | 特点 | 适合 |
|------|------|------|------|
| 初级 | 20 以内 | 不进位不退位 | 幼儿园大班 |
| 中级 | 50以内 | 有进位退位 | 幼小衔接 |
| 高级 | 100以内 | 混合运算 | 一年级上学期 |
| 两位数 | 10-99 | 两位数运算 | 一年级下学期 |
| 三位数 | 100-999 | 三位数运算 | 二年级 |

### 🎯 专项练习（8 种模式）
- **混合运算** — 随机生成连加、连减、加减混合
- **连加专项** — 专门练习 3 个数相加
- **连减专项** — 专门练习 3 个数相减
- **进位加法** — 集中训练进位技巧
- **退位减法** — 集中训练退位技巧
- **两位数加法** — 两位数加法专项（10-99）
- **两位数减法** — 两位数减法专项（10-99）
- **三位数混合** — 三位数加减混合（100-999）

### 🎨 A4 智能排版
- **标准布局**：5 列 × 10 行 = 50 题/页
- **多页支持**：最多 5 页（250 题）
- **每页配色**：蓝→绿→暖→紫→灰，5 种颜色方案
- **页间分隔**：自动分隔线 + 分页符
- **大字号**：微软雅黑 14pt，易于阅读
- **柔和配色**：保护儿童视力

### 🔒 专业功能
- **答案深度隐藏** — 使用 `xlSheetVeryHidden`，防止孩子偷看
- **练习记录** — 自动记录每次练习的日期、难度、模式、题数
- **负数控制** — 可精确控制负数出现概率（建议 0）
- **一键导出 PDF** — 快速生成可打印的 PDF 文件

## 🚀 快速开始

### 第 1 步：准备 Excel 文件
新建 Excel 工作簿，保存为 **Excel 启用宏的工作簿 (*.xlsm)**

### 第 2 步：粘贴 VBA 代码
按 `Alt + F11` 打开 VBA 编辑器 → 插入模块 → 粘贴 `src/VBA 代码/完整 VBA 代码.bas` 中的代码

### 第 3 步：创建按钮
启用「开发工具」选项卡 → 插入 6 个表单控件按钮，分别绑定对应宏

> 详细步骤请查看 [安装指南.md](docs/安装指南.md)

## 🎯 核心优势

- **专业化** — 针对幼升小儿童设计，符合教育规律
- **智能化** — 根据难度自动调整参数，无需手动设置
- **人性化** — 柔和配色保护视力，大字号易于阅读
- **便捷化** — 一键生成，一键打印，一键导出
- **系统化** — 自动记录练习历史，追踪学习进步
- **安全化** — 答案深度隐藏，防止孩子偷看
- **标准化** — A4 标准排版，5列×10行=50题/页
