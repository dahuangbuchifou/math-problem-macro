# 数学题生成宏命令

## 项目简介

数学题生成宏命令，用于快速生成各类数学题目。支持多种题型、难度分级、自动排版，适用于教学、练习、考试等场景。

## 功能特性

- 🎲 **随机出题** — 支持加减乘除、分数、方程等多种题型
- 📊 **难度分级** — 小学、初中、高中，按需选择
- 📝 **自动排版** — 生成格式规范的试卷/练习题
- ⚡ **批量生成** — 一次生成多套题目，满足教学需求
- 🔧 **灵活配置** — 题目数量、题型比例、参数范围均可自定义
