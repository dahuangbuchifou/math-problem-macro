**代码审查：**
```vba
' 两位数加法：a ∈ [10,99], b ∈ [1, 99-a], 确保 a+b ≤ 99
a = Int(Rnd() * 90) + 10
b = Int(Rnd() * (99 - a + 1)) + 1

' 两位数减法：a ∈ [10,99], b ∈ [10, a-1], 确保 a-b ≥ 1
a = Int(Rnd() * 90) + 10
b = Int(Rnd() * (a - 10)) + 10
```
✅ 逻辑正确，边界条件处理完善。

| 检查项 | 预期 | 实际 | 结果 |

| 下拉菜单新增「三位数混合」 | H6 包含此选项 | ✅ 已添加 | 通过 |
| 题目范围 100-999 | a,b ∈ [100,999] | ✅ GenerateThreeDigitMixed 正确 | 通过 |
| 支持加减混合 | 随机选择 + 或 - | ✅ Rnd() > 0.5 分支 | 通过 |
| 减法结果非负 | a - b ≥ 0 | ✅ b < a 确保 | 通过 |

**代码审查：**
```vba
a = Int(Rnd() * 900) + 100  ' 100-999
If Rnd() > 0.5 Then
    b = Int(Rnd() * (999 - a + 1)) + 1  ' 加法：a+b ≤ 999
Else
    b = Int(Rnd() * (a - 100)) + 100    ' 减法：b ≥ 100, b < a
End If
```
✅ 逻辑正确，加减法分支清晰。
