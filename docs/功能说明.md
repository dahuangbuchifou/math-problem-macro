# ⚙️ 功能说明

> 6 个功能按钮 + 参数设置面板 + 多页生成，全面解析 V2.0

---

## 🎛️ 功能按钮一览

### 按钮 1：生成题目
- **宏名称**：`GenerateQuestions`
- **位置**：K1
- **功能**：一键生成数学练习题
- **详细说明**：
  - 根据 H5（难度等级）、H6（练习模式）、H7（页数）自动生成题目
  - 题目生成在「题目页」工作表
  - 答案自动生成在「答案页」工作表（默认隐藏）
  - 自动记录练习信息到「练习记录」工作表
  - **A4 标准排版**：5 列 × 10 行 = 50 题/页
  - **多页支持**：最多 5 页，每页不同颜色方案
  - **页间分隔**：自动添加分隔线和分页符
  - 每 5 题轮换柔和颜色（每页独立配色）

### 按钮 2：显示/隐藏答案
- **宏名称**：`ToggleAnswerSheet`
- **位置**：K5
- **功能**：切换答案页的可见性
- **详细说明**：
  - 第一次点击：显示「答案页」
  - 再次点击：重新隐藏「答案页」
  - 答案页使用深度隐藏（xlSheetVeryHidden），防止孩子通过右键取消隐藏

### 按钮 3：打印预览
- **宏名称**：`PrintPreviewSheet`
- **位置**：K9
- **功能**：预览打印效果
- **详细说明**：
  - 打开打印预览窗口
  - 可查看题目排版效果
  - 多页时自动分页预览
  - 确认无误后直接打印

### 按钮 4：重置设置
- **宏名称**：`ResetSettings`
- **位置**：K13
- **功能**：恢复所有参数为默认值
- **详细说明**：
  - 最大数字范围 → 100
  - 题目数量 → 50
  - 负数概率 → 0%
  - 难度等级 → 中级
  - 练习模式 → 混合运算
  - 生成页数 → 1
  - 当前状态 → 就绪

### 按钮 5：查看记录
- **宏名称**：`ViewRecords`
- **位置**：K17
- **功能**：查看练习历史记录
- **详细说明**：
  - 显示「练习记录」工作表
  - 记录包含：日期时间、难度、模式、题目数量
  - 方便追踪孩子的学习进度

### 按钮 6：导出PDF
- **宏名称**：`ExportToPDF`
- **位置**：K21
- **功能**：一键导出可打印的 PDF 文件
- **详细说明**：
  - 将题目页导出为 PDF
  - 自动命名（含日期和难度）
  - 导出后自动打开所在文件夹

## 📊 参数设置面板

参数面板位于 **G1:H7** 区域：

### G1/H1：最大数字范围
- **说明**：题目中出现的最大数字
- **默认值**：100
- **注意**：选择难度等级时会自动设置此值

### G2/H2：题目数量
- **说明**：一次生成的题目数量
- **默认值**：50
- **范围**：1 - 250（5 页 × 50 题）
- **注意**：A4 纸一页 50 题（5列×10行）

### G3/H3：负数概率 (%)
- **说明**：出现负数的概率
- **默认值**：0（不出现负数）
- **建议**：幼升小阶段设为 0

### G4/H4：当前状态
- **说明**：显示当前工具状态
- **只读**：不可手动修改
- **状态值**：就绪 / 生成中 / 已完成 / 错误

### G5/H5：难度等级（下拉菜单）

| 难度 | 范围 | 特点 | 适合 |
|------|------|------|------|
| **初级** | 20 以内 | 不进位、不退位 | 幼儿园大班 |
| **中级** | 50以内 | 有进位、有退位 | 幼小衔接 |
| **高级** | 100以内 | 混合运算 | 一年级上学期 |
| **两位数** | 10-99 | 两位数运算 | 一年级 |
| **三位数** | 100-999 | 三位数运算 | 二年级 |

### G6/H6：练习模式（下拉菜单）

| 模式 | 说明 | 示例 |
|------|------|------|
| **混合运算** | 随机生成加减法、连加连减 | `3 + 5 =`、`8 - 3 =`、`2 + 3 + 4 =` |
| **连加专项** | 专门练习 3 个数相加 | `5 + 3 + 2 =` |
| **连减专项** | 专门练习 3 个数相减 | `10 - 3 - 2 =` |
| **进位加法** | 专门练习进位加法 | `28 + 5 =`（个位 8+5=13 进位） |
| **退位减法** | 专门练习退位减法 | `13 - 8 =`（个位 3-8 退位） |
| **两位数加法** | 两位数加法专项 | `25 + 37 =` |
| **两位数减法** | 两位数减法专项 | `62 - 38 =` |
| **三位数混合** | 三位数加减混合 | `356 + 241 =`、`500 - 123 =` |

### G7/H7：生成页数（下拉菜单）

| 页数 | 题目数 | 说明 |
|------|--------|------|
| **1** | 50 题 | 默认，A4 一页 |
| **2** | 100 题 | A4 两页 |
| **3** | 150 题 | A4 三页 |
| **4** | 200 题 | A4 四页 |
| **5** | 250 题 | 最大，A4 五页 |

**多页特性：**
- 每页使用不同的颜色方案（蓝→绿→暖→紫→灰）
- 页与页之间有分隔线
- 自动插入分页符，打印时自动分页
- 每页 50 题（5 列 × 10 行）

## 📐 A4 排版系统

### 标准布局
- **纸张**：A4 纵向
- **列数**：5 列（列宽 20）
- **行数**：10 行/页（行高 28）
- **每页题数**：5 × 10 = 50 题
- **字体**：微软雅黑 14pt
- **对齐**：水平垂直居中

### 多页布局
- **分隔行**：页间插入 1 行分隔线（行高 15）
- **分页符**：自动插入水平分页符
- **总行数**：页数 × 11 - 1（含分隔行）
- **打印适配**：自动缩放适配 A4

### 示例：3 页布局
```
行 1-10:  第 1 页（50 题，蓝色系）
行 11:     分隔线
行 12-21: 第 2 页（50 题，绿色系）
行 22:     分隔线
行 23-32: 第 3 页（50 题，暖色系）
```

## 🔒 安全特性

- **答案深度隐藏**：使用 `xlSheetVeryHidden`，无法通过右键取消隐藏
- **负数控制**：可精确控制负数出现概率（建议 0）
- **数据保护**：生成题目前自动清除旧数据
