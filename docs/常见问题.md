# ❓ 常见问题

> 使用中遇到的问题，这里都有答案（V2.0）

---

## 安装相关

### Q1：保存时找不到「启用宏的工作簿」选项？
**A**：在「另存为」对话框中，点击「保存类型」下拉框，选择「Excel 启用宏的工作簿 (*.xlsm)」。如果没有看到，确保你的 Excel 版本是 2007 或以上。

### Q2：VBA 编辑器打不开（Alt+F11 没反应）？
**A**：
- 确认你使用的是完整版 Microsoft Excel（不是 WPS 或在线版）
- 尝试：文件 → 选项 → 加载项 → 管理：COM 加载项 → 确保没有禁用 VBA
- 某些精简版 Office 可能不支持 VBA，建议安装完整版

### Q3：粘贴代码后提示语法错误？
**A**：
- 检查是否完整粘贴了所有代码（约 900 行）
- 确保粘贴到的是「模块」中，不是「ThisWorkbook」或工作表代码区
- 检查是否有中文字符被意外修改
- V2.0 代码已修复所有语法错误

## 使用相关

### Q4：点击「生成题目」没反应？
**A**：
1. 检查是否已启用宏（打开文件时是否点击了「启用内容」）
2. 确认 VBA 代码已正确粘贴到模块中
3. 按 `Alt+F11` 打开编辑器，按 `F5` 运行 `GenerateQuestions` 看是否有报错
4. 检查 H5、H6、H7 下拉菜单是否已正确设置

### Q5：题目生成后格式不对？
**A**：
- 确保「题目页」工作表名称正确（不是 Sheet1）
- 尝试点击「重置设置」后重新生成
- 检查是否有其他内容占用了题目区域
- V2.0 使用标准 A4 排版（5列×10行），如果格式不对可能是字体未安装

### Q6：找不到答案页？
**A**：答案页默认是**深度隐藏**的（防止孩子偷看）。点击「**显示/隐藏答案**」按钮即可显示。无法通过右键「取消隐藏」找到答案页，这是正常的安全设计。

### Q7：导出 PDF 失败？
**A**：
- 确保已生成题目（先生成题目，再导出）
- 检查是否有足够的磁盘空间
- 确认 Excel 版本支持 PDF 导出功能（2007 及以上）
- 尝试手动导出：文件 → 另存为 → 选择 PDF 格式

### Q8：打印效果不理想？
**A**：
- 先点击「打印预览」查看效果
- 在打印设置中调整：纸张大小（A4）、方向（纵向）、边距
- 建议：页边距适中，居中方式勾选「水平」
- V2.0 已自动设置 A4 纵向 + 水平居中，通常无需额外调整
- 多页时会自动分页，每页 50 题

## 功能相关

### Q9：初级/中级/高级/两位数/三位数有什么区别？
**A**：
| 难度 | 数字范围 | 特点 | 适合年龄 |
|------|---------|------|---------|
| 初级 | 20 以内 | 不进位、不退位 | 幼儿园大班 |
| 中级 | 50以内 | 有进位、有退位 | 幼小衔接 |
| 高级 | 100以内 | 混合运算 | 一年级上学期 |
| 两位数 | 10-99 | 两位数运算 | 一年级下学期 |
| 三位数 | 100-999 | 三位数运算 | 二年级 |

### Q10：负数概率是什么意思？
**A**：控制题目结果出现负数的概率。幼升小阶段建议设为 0，不出现负数。

### Q11：练习记录在哪里？
**A**：点击「查看记录」按钮，会显示「练习记录」工作表。每次生成题目时会自动记录：日期时间、难度等级、练习模式、题目数量。

### Q12：可以自定义题目数量吗？
**A**：可以。在 H2 单元格输入题目数量，或在 H7 选择页数（1-5 页）。每页 50 题，最多 250 题。

### Q13：多页生成怎么用？
**A**：
1. 在 H7 选择页数（如 3）
2. 点击「生成题目」
3. 系统会自动生成 3 页（150 题），每页 50 题
4. 每页使用不同颜色（蓝→绿→暖）
5. 页间有分隔线和分页符
6. 打印预览确认分页效果

### Q14：每页的颜色为什么不一样？
**A**：V2.0 为每页设计了独立的颜色方案，方便区分页面：
- 第 1 页：蓝色系
- 第 2 页：绿色系
- 第 3 页：暖色系
- 第 4 页：紫色系
- 第 5 页：中性色系

### Q15：8 种练习模式怎么选？
**A**：
| 模式 | 适用场景 |
|------|---------|
| 混合运算 | 日常综合练习 |
| 连加专项 | 训练多个数连加 |
| 连减专项 | 训练多个数连减 |
| 进位加法 | 孩子进位不熟练时 |
| 退位减法 | 孩子退位不熟练时 |
| 两位数加法 | 两位数加法专项训练 |
| 两位数减法 | 两位数减法专项训练 |
| 三位数混合 | 三位数加减法训练 |

## 安全相关

### Q16：宏安全性警告怎么解决？
**A**：
1. 文件 → 选项 → 信任中心 → 信任中心设置
2. 宏设置 → 选择「启用所有宏」（不推荐）或「为所有宏显示警告」（推荐）
3. 或者将文件保存到「受信任的位置」

### Q17：文件可以分享给其他人使用吗？
**A**：可以。将 `.xlsm` 文件复制给他人，对方打开时启用宏即可使用。V2.0 打开文件后会自动初始化参数面板和下拉菜单。

### Q18：孩子不小心显示了答案页怎么办？
**A**：点击「显示/隐藏答案」按钮可以重新隐藏。答案页使用深度隐藏，孩子无法通过常规方式（右键取消隐藏）找到答案页。

## 其他

### Q19：支持 WPS 吗？
**A**：WPS 的 VBA 兼容性有限，部分功能可能无法正常运行。建议使用 Microsoft Office Excel 2007 或以上版本。

### Q20：可以生成乘法/除法题目吗？
**A**：当前版本专注于加减法（幼升小需求）。乘法/除法功能可在后续版本中添加。

### Q21：如何备份题目？
**A**：
- 方法一：导出 PDF 存档
- 方法二：另存为新的 `.xlsm` 文件
- 方法三：复制题目页到新的工作簿

### Q22：生成题目很慢怎么办？
**A**：
- 减少题目数量或页数
- 关闭其他占用资源的程序
- 检查电脑性能（建议 4GB 以上内存）
- V2.0 已优化生成逻辑，速度比 V1.0 更快

### Q23：有 bug 或建议怎么反馈？
**A**：欢迎通过项目仓库提交 Issue，或联系工部团队。
