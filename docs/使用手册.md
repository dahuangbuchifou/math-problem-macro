# 📖 使用手册

> 日常使用流程，从选题到打印一站式指南（V2.0）

---

## 🎯 快速上手：3 步生成题目

1. **选择难度**：在 H5 下拉菜单选择（初级 / 中级 / 高级 / 两位数 / 三位数）
2. **选择模式**：在 H6 下拉菜单选择练习模式
3. **点击生成**：点击「**生成题目**」按钮

完成！题目自动生成在「题目页」，答案自动生成在「答案页」。

---

## 📋 日常使用流程

### 第一步：设置参数

| 参数 | 位置 | 说明 |
|------|------|------|
| 难度等级 | H5 | 初级(20) / 中级(50) / 高级(100) / 两位数(10-99) / 三位数(100-999) |
| 练习模式 | H6 | 8 种模式可选 |
| 题目数量 | H2 | 一次生成的题目数，默认 50 |
| 负数概率 | H3 | 建议设为 0 |
| 生成页数 | H7 | 1-5 页，默认 1 页（50 题） |

### 第二步：生成题目

点击「**生成题目**」按钮，系统会自动：
- ✅ 根据难度和模式生成题目
- ✅ A4 标准排版（5列×10行）
- ✅ 每页不同颜色方案
- ✅ 页间自动分隔线 + 分页符
- ✅ 生成隐藏的答案页
- ✅ 记录练习信息

### 第三步：打印 / 导出

**方式一：直接打印**
- 点击「**打印预览**」查看效果
- 确认无误后点击「打印」

**方式二：导出 PDF**
- 点击「**导出PDF**」按钮
- 选择保存位置
- 生成可打印的 PDF 文件

### 第四步：查看答案

- 点击「**显示/隐藏答案**」按钮
- 答案页会显示/隐藏切换
- 建议孩子做完题后再显示答案

## 🎓 难度选择建议

### 初级（20 以内）
- **适合**：幼儿园大班（5-6 岁）
- **特点**：不进位、不退位
- **推荐题目数**：30-50 题
- **推荐模式**：混合运算

### 中级（50以内）
- **适合**：幼小衔接（6-7 岁）
- **特点**：有进位、有退位
- **推荐题目数**：50 题（1 页）
- **推荐模式**：进位加法 / 退位减法

### 高级（100以内）
- **适合**：一年级上学期
- **特点**：混合运算、难度较高
- **推荐题目数**：50-100 题（1-2 页）
- **推荐模式**：混合运算 / 连加专项 / 连减专项

### 两位数（10-99）
- **适合**：一年级下学期
- **特点**：两位数运算，支持进位/退位
- **推荐题目数**：50 题（1 页）
- **推荐模式**：两位数加法 / 两位数减法

### 三位数（100-999）
- **适合**：二年级
- **特点**：三位数加减法
- **推荐题目数**：50 题（1 页）
- **推荐模式**：三位数混合

## 📊 多页生成指南

### 何时使用多页？
- 需要大量练习时（如周末集中训练）
- 班级统一出题时
- 需要存档不同日期的题目时

### 多页颜色区分
| 页码 | 颜色主题 | 视觉效果 |
|------|---------|---------|
| 第 1 页 | 蓝色系 | 清新淡雅 |
| 第 2 页 | 绿色系 | 自然舒适 |
| 第 3 页 | 暖色系 | 温暖活泼 |
| 第 4 页 | 紫色系 | 优雅柔和 |
| 第 5 页 | 中性色系 | 简洁素雅 |

### 打印多页题目
1. 在 H7 选择页数（如 3）
2. 点击「生成题目」
3. 点击「打印预览」确认分页效果
4. 直接打印或导出 PDF

## 📊 练习记录查看

点击「**查看记录**」按钮，可以看到：
- 📅 练习日期和时间
- 📈 难度等级和练习模式
- 🔢 题目数量
- 📄 工作表状态

记录保存在「练习记录」工作表中，方便追踪学习进度。

---

## 🔄 重置设置

如果参数设置乱了，点击「**重置设置**」按钮：
- 难度等级 → 中级
- 练习模式 → 混合运算
- 题目数量 → 50
- 负数概率 → 0%
- 生成页数 → 1
- 状态 → 就绪

## 💡 使用技巧

1. **一页刚好 50 题**：A4 纸一页正好 50 题（5列×10行），打印效果最佳
2. **专项突破**：孩子某类题薄弱，用专项模式集中练习
3. **多页批量**：一次生成多页，分天使用
4. **答案管理**：生成题目后答案自动隐藏，防止孩子偷看
5. **PDF 存档**：导出的 PDF 可以按日期命名存档，方便回顾
6. **颜色区分**：多页生成时不同颜色帮助区分页面

---

## ⚠️ 注意事项

- 每次生成题目会**覆盖**之前的题目，请及时打印或导出
- 答案页默认**深度隐藏**，需点击按钮才能查看
- 导出 PDF 时确保已生成题目
- 宏安全性设置需允许运行宏
- 多页生成时建议先预览再打印
